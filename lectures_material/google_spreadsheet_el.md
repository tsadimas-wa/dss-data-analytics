# Οδηγός Live Demo: OLAP Operations στο Google Sheets

**Dataset:** Superstore Sales  
**Εργαλείο:** Google Sheets (Συγκεντρωτικοί Πίνακες / Pivot Tables)

---

## Βήμα 0: Λήψη του Dataset από το Kaggle

Πριν ξεκινήσουμε, χρειαζόμαστε το αρχείο δεδομένων **Superstore Sales**. Ακολουθήστε τα παρακάτω βήματα για να το κατεβάσετε από το Kaggle.

### Δημιουργία Λογαριασμού στο Kaggle (αν δεν έχετε)

1. Μεταβείτε στη σελίδα [www.kaggle.com](https://www.kaggle.com).
2. Πατήστε **Register** (πάνω δεξιά) και δημιουργήστε δωρεάν λογαριασμό (με email ή Google account).
3. Επιβεβαιώστε το email σας αν σας ζητηθεί.

### Λήψη του αρχείου CSV

1. Συνδεθείτε στο Kaggle και αναζητήστε **"Superstore Sales Dataset"** στη γραμμή αναζήτησης.
2. Επιλέξτε το dataset **"Sample - Superstore"** (ή αναζητήστε απευθείας: `kaggle datasets search superstore`).
3. Στη σελίδα του dataset, πατήστε την καρτέλα **Data**.
4. Κατεβάστε το αρχείο `Sample - Superstore.csv` (ή `superstore.csv`) πατώντας το κουμπί **Download**.

> 💡 **Εναλλακτικά:** Μπορείτε να κατεβάσετε το dataset απευθείας από τον σύνδεσμο που σας έχει δώσει ο εισηγητής του μαθήματος.

### Εισαγωγή στο Google Sheets

1. Μεταβείτε στο [Google Sheets](https://sheets.google.com) και δημιουργήστε ένα **νέο spreadsheet**.
2. Πηγαίνετε στο μενού: **Αρχείο > Εισαγωγή** (File > Import).
3. Επιλέξτε **Μεταφόρτωση** (Upload) και σύρετε το αρχείο `.csv` που κατεβάσατε.
4. Στις επιλογές εισαγωγής:
   - **Τύπος διαχωριστή:** Κόμμα (Comma)
   - **Μετατροπή κειμένου σε αριθμούς:** Ναι
5. Πατήστε **Εισαγωγή δεδομένων** (Import data).

> ✅ Αν το αρχείο φορτώθηκε σωστά, θα δείτε στήλες όπως: `Order ID`, `Category`, `Sub-Category`, `Sales`, `Region`, `Segment`, `Ship Mode`, κ.ά.

## Βήμα 1: Η Προετοιμασία (Δημιουργία του Κύβου)

1. Ανοίξτε το Google Sheet με τα δεδομένα **Superstore** που εισαγάγατε.
2. Κάντε κλικ σε οποιοδήποτε κελί μέσα στα δεδομένα.
3. Πατήστε `Ctrl + A` (ή `Cmd + A` σε Mac) για να επιλέξετε όλο τον πίνακα.
4. Πηγαίνετε στο μενού: **Εισαγωγή > Συγκεντρωτικός Πίνακας** (Insert > Pivot table) και επιλέξτε δημιουργία σε **"Νέο φύλλο"**.
5. Στο δεξί μενού που θα ανοίξει (**Επεξεργαστής συγκεντρωτικού πίνακα**):
   - Στις **Γραμμές (Rows)** → προσθέστε το `Category`
   - Στις **Στήλες (Columns)** → προσθέστε το `Region`
   - Στις **Τιμές (Values)** → προσθέστε το `Sales` *(βεβαιωθείτε ότι η συνάρτηση είναι **SUM / Άθροισμα**)*

> 🗣️ **Τι να πείτε:** *"Αυτή τη στιγμή φτιάξαμε έναν απλό 2D πίνακα. Όμως στο παρασκήνιο, το εργαλείο έχει φορτώσει όλες τις διαστάσεις σαν έναν Κύβο του Ρούμπικ. Πάμε να τον κινήσουμε!"*

## Βήμα 2: Τα OLAP Operations (Το Live Demo)

> **Σημείωση:** Κάθε operation εκτελείται μέσα στον ίδιο Pivot Table. Δεν χρειάζεται να δημιουργείτε νέο φύλλο κάθε φορά.

---

### 1. PIVOT (Περιστροφή)

| | |
|---|---|
| **Τι κάνετε** | Πιάστε με το ποντίκι το `Category` από τις Γραμμές και σύρετέ το στις Στήλες. Πιάστε το `Region` και βάλτε το στις Γραμμές. |
| **Αποτέλεσμα** | Ο πίνακας "αναποδογυρίζει" — οι Περιοχές γίνονται γραμμές και οι Κατηγορίες στήλες. |

> 🗣️ **Τι να πείτε:** *"Μόλις κάναμε Pivot! Αλλάξαμε την οπτική μας γωνία. Τα δεδομένα είναι τα ίδια, αλλά τώρα βλέπουμε τις Περιοχές ανά Κατηγορία, αντί για τις Κατηγορίες ανά Περιοχή."*

---

### 2. DRILL-DOWN (Εμβάθυνση στη λεπτομέρεια)

| | |
|---|---|
| **Τι κάνετε** | Επαναφέρετε το `Category` στις Γραμμές. Στη συνέχεια, πατήστε **"Προσθήκη"** στις Γραμμές και προσθέστε **κάτω** από το `Category` το `Sub-Category`. |
| **Αποτέλεσμα** | Κάθε κατηγορία (π.χ. Furniture) αναλύεται στα επιμέρους υπο-προϊόντα της (Chairs, Tables, κλπ.). |

> 🗣️ **Τι να πείτε:** *"Κάναμε Drill-down — ζουμάραμε! Αντί να βλέπουμε γενικά τα 'Έπιπλα', τώρα βλέπουμε πόσο πούλησαν οι 'Καρέκλες' και τα 'Τραπέζια' ξεχωριστά."*

---

### 3. ROLL-UP (Σύνοψη / Μεγάλη Εικόνα)

| | |
|---|---|
| **Τι κάνετε** | Διαγράψτε το πεδίο `Sub-Category` από το δεξί μενού (πατήστε το **✕** δίπλα του), ή κλείστε τα `+` δίπλα σε κάθε κατηγορία. |
| **Αποτέλεσμα** | Επιστρέφουμε στη συνολική εικόνα ανά κατηγορία — η λεπτομέρεια κρύβεται. |

> 🗣️ **Τι να πείτε:** *"Η αντίστροφη πορεία είναι το Roll-up. Κρύβουμε τη λεπτομέρεια για να δούμε τη συνολική, μεγάλη εικόνα (macro level)."*

---

### 4. SLICE (Τεμαχισμός)

| | |
|---|---|
| **Τι κάνετε** | Πηγαίνετε στην ενότητα **Φίλτρα (Filters)** στο δεξί μενού. Προσθέστε το `Segment`. Πατήστε πάνω του, ξε-τικάρετε όλα και αφήστε επιλεγμένο **ΜΟΝΟ** το `Corporate`. |
| **Αποτέλεσμα** | Ο πίνακας εμφανίζει δεδομένα αποκλειστικά για εταιρικούς πελάτες. |

> 🗣️ **Τι να πείτε:** *"Μόλις κόψαμε μια 'φέτα' από τον κύβο (Slice). Αγνοήσαμε τους απλούς καταναλωτές και βλέπουμε την ίδια ανάλυση ΜΟΝΟ για τους εταιρικούς πελάτες."*

---

### 5. DICE (Κυβισμός / Υπο-κύβος)

| | |
|---|---|
| **Τι κάνετε** | Προσθέστε ένα **δεύτερο** φίλτρο: το `Ship Mode` (Τρόπος Αποστολής). Αφήστε τικαρισμένο **ΜΟΝΟ** το `First Class`. |
| **Αποτέλεσμα** | Ο πίνακας δείχνει τώρα δεδομένα για **Corporate πελάτες που επέλεξαν First Class αποστολή** — ένας "κύβος μέσα στον κύβο". |

> 🗣️ **Τι να πείτε:** *"Όταν συνδυάζουμε πολλαπλά φίλτρα ταυτόχρονα, κάνουμε Dice! Δημιουργήσαμε έναν υπο-κύβο: βλέπουμε πωλήσεις ΜΟΝΟ για Corporate πελάτες ΠΟΥ επέλεξαν First Class αποστολή."*
